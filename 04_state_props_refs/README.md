## react 组件的三大属性

#### state 组件对象 最重要的属性
```
    1. 值是一个对象（可以包含多个数据）, 所以也称 state 为 状态机
```

#### props
```
    props
```

#### refs 与 事件处理
```
    refs 保存了组件类 所有含有 ref 属性的标签元素
```

一、props 和 state 的区别

1. 相同点
    都是存数据的

2. 不同点
    state 是组件内部可变化的数据
    props 是从组件外部向组件内部传递的 数据，只读，属性不可修改
